# 🖥 YmlY 螢幕錄影 YmlY Screen Recorder

## 功能特色

* 透過瀏覽器錄製螢幕畫面

* 可同時錄製系統聲音（僅限 Chrome）、麥克風聲音

* 可以倒數計時後錄影

* 支援 Windows、MacOS 版 Chrome、Firefox 瀏覽器

    * ***不支援支援行動裝置瀏覽器、Safari 瀏覽器***

## 使用方式

* 線上使用：[https://ottokang.github.io/YmlY-Screen-Recorder/](https://ottokang.github.io/YmlY-Screen-Recorder/)

## YmlY 是什麼？

* 螢（Y）幕（m）錄（l）影（Y）的漢語拼音縮寫
